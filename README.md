## Setup

The below steps assume that you PNPM, Node and Git installed on your machince.

```sh
# Clone the repo
git clone https://github.com/hammadmajid/website.astro

# Install the dependencies
pnpm install

# Build for production
pnpm build

# Stat a dev server
pnpm dev
```
